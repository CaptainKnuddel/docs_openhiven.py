Class HivenClient¶

HivenClient¶

class openhivenpy.Client.HivenClient(args*, kwargs**)¶

Main Class used to interact with the Websocket and the Hiven API. Inherited by Bot Client and User Client

Parameters

  • token (int) – Token for the interaction with Hiven.

  • heartbeat – Hearbeat interval for sending messages to the Hiven Websocket

  • client_type (str) –

    Type of the HivenClient. Can only be either “user”, “bot” or None.

    Note

    Will be automatically set if Bot Client or User Client is used! If the HivenClient: HivenClient is directly used as an object, it is recommended to set it to None!
