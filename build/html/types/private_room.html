Private Room¶

class PrivateRoom¶

Represents a private message room with another user or bot

Note

This is not to be confused with PrivateGroupRoom! PrivateGroupRoom is for private non-group rooms, whilst PrivateRoom is private non-group rooms.

user¶

The user in the room

recipient¶

Aliases of user

id¶

The ID of the room

last_message_id¶

The last message’s ID

name¶

The name of the room

send(content, delay)¶

This function is a coroutine!

Sends a message in the room

Parameters

  • content (str) – The message’s content

  • attatchment (binary) – The message’s attachment

  • delay (float) – Delay until execution

Returns

The Message object if successful else None

Message
