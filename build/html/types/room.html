Room¶

class Room¶

Represents a Hiven Room in a House

id¶

Returns the room ID

name¶

Returns the rooms name

house¶

Returns the house the room belongs in

position¶

Returns the position of the room in the category

emoji¶

Returns the room’s emoji

description¶

Returns the room’s description

send(kwargs)¶

Sends a message in the Room. Returns the message if successful.

Async

Parameters

  • content (str) – The message’s content

  • delay (float) – Delay until execution. Defaults to 0

Returns

The message object if successful else None

Return type

Message

edit(**kwargs)¶

Edits the Room. The client needs permissions to do this.

Async

Parameters

  • name (str) – The room name

  • emoji (str) – The room’s emoji

  • description (str) – The room’s description

Returns

True if successful else False

Return type

bool
