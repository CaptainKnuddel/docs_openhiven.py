Room¶

id¶

Returns the room ID

name¶

Returns the rooms name

house¶

Returns the house the room belongs in

position¶

Returns the position of the room in the category

emoji¶

Returns the room’s emoji

description¶

Returns the room’s description

send(kwargs)¶

Sends a message in the Room. Returns the message if successful.

Parameters

  • content (string) – The content of the message

  • delay (float) – The delay before sending the message. Defaults to 0.

  • attatchment (file) – The attatchment to send with the message. Defaults to None

Returns

Message

edit(kwargs)¶

Edits the Room. The client needs permissions to do this. Returns True if successful.

Parameters

  • name (string) – The room name

  • emoji (string) – The room’s emoji

  • description (string) – The room’s description
