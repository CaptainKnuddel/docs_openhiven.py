Message¶

class Message¶
id¶

Returns the ID of the message.

author¶

Returns the author of the message.

created_at¶

Returns the time the message was sent.

edited_at¶

Returns the time the message was edited. Will return None if it hasn’t been edited.

room¶

Returns the Room the message belongs to

house¶

Returns the House the message belongs to. May return None in a Private Room

attachment¶

Returns the Attachment of the message. May return None.

content¶

Returns the message’s content.

mentions¶

Returns the list of mentions in the message. May return None.

embed¶

Returns the embed in the message.

mark_message_as_read(delay)¶

Marks the message as read. Bot clients do not need to use this.

Async

Parameters

delay (float) – Delay until execution

Returns

True if successful else False

Return type

bool

delete(delay=0)¶

Deletes the message. Raises Forbidden if not allowed.

Async

Parameters

delay (float) – Delay until execution

Returns

The DeletedMessage object if successful else False

Return type

DeletedMessage

edit(content)¶

Edits the message. Returns True if successful.

Parameters

content (str) – The message’s content.

Return type

bool
